## Creating Fake Page using ( Zphisher )

### Step 1 ( installing Zphisher )

```shell

git clone https://github.com/htr-tech/zphisher.git

```

### Step 2 ( running the program )

```shell

cd zphisher

bash zphisher.sh

```
